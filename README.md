# ⚛️ React To-Do List App

A simple and responsive **To-Do List application** built using **React + Vite**.
This app helps users manage daily tasks efficiently by adding and deleting tasks in real time.

---

## 🌐 Live Demo

👉 [Try it here](https://react-to-do-list-git-main-mohit823s-projects.vercel.app/)

---

## 🚀 Features

* ➕ Add new tasks
* ❌ Delete tasks
* ⌨️ Press Enter key to add task
* 🧠 Input validation (no empty tasks)
* ⚡ Instant UI updates using React state
* 📱 Responsive UI

---

## 🛠️ Technologies Used

* ⚛️ React (Hooks – `useState`)
* ⚡ Vite (Fast build tool)
* 🎨 CSS

## ⚙️ How It Works

* Tasks are stored using **React state (`useState`)**

### Actions:

* **Add Button क्लिक** → Task list में add होता है
* **Enter key press** → Task add होता है
* **Delete button** → Task remove हो जाता है

1. Clone the repository:

```bash id="4ct54e"

```

2. Go to project folder:



4. Run the app:
